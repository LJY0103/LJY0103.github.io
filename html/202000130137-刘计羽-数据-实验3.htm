实验题目：利用云平台搭建个人博客

实验学时：2

实验日期：    2023/3/19

实验目的：熟悉个人博客系统的搭建。

具体包括：

参考方案：注册Github账号，搭建Hexo环境并实现个人博客搭建，撰写实验报告。

 

 

硬件环境： 

联网的计算机一台

 

 

软件环境：

Windows or Linux

 

 

实验步骤与内容：

1.  下载安装node.js、git

检查环境是否配置成功

2.  注册GitHub账号，设置用户名及邮箱。

3.  使用git bash设置用户名和邮箱

git config --global user.name "GitHub 用户名"

git config --global user.email "GitHub 邮箱"

4.  创建SSH密匙

ssh-keygen -t rsa -C "GitHub 邮箱"

5.  复制公钥到GitHub的key中。

6.  检验是否可以成功连接

7.  安装hexo

npm install -g hexo-cli

8.  初始化hexo

hexo init

npm install

hexo g

hexo s

9.  安装hexo-deployer-git

npm install hexo-deployer-git –save

10.修改_config.yml

添加：

deploy:

  type: git

  repository: git@github.com:用户名/用户名.github.io.git

  branch: master

11.执行hexo d将网站部署到github。
